# Add 2 Numbers (Java)

## Description
Cộng 2 số lớn được biểu diễn dưới dạng chuỗi.
Thuật toán mô phỏng cách cộng của học sinh tiểu học (có nhớ).

---

## How it works
- Duyệt từ phải sang trái
- Cộng từng chữ số
- Nếu >= 10 thì nhớ (carry)
- Lưu kết quả và đảo ngược lại

---

## Run

### Compile
javac Main.java

### Run
java Main

---

## Example

Input:
1234 + 897

Output:
2131

---

## Requirements
- Java JDK 8+
